        <h2>About the Project</h2>
        <p>
            Pharma Chat AI is an AI-powered chat application designed to provide quick and accurate responses to pharmaceutical queries.
            The application leverages modern AI models to analyze data and generate insightful responses.
        </p>
        <h2>Getting Started</h2>
        <h3>1. Clone the Repository</h3>
        <pre>
git clone https://github.com/AwadeYuvraj/Pharma_Chat_AI.git
cd pharma-chat-ai
        </pre>
        <h3>2. Create a Virtual Environment</h3>
        <pre>
python -m venv venv
        </pre>
        <h3>3. Activate the Virtual Environment</h3>
        <p>Run the appropriate command based on your operating system:</p>
        <ul>
            <li><code>source venv/bin/activate</code> (Linux/Mac)</li>
            <li><code>venv\Scripts\activate</code> (Windows Command Prompt)</li>
            <li><code>.\venv\Scripts\Activate.ps1</code> (Windows PowerShell)</li>
        </ul>
        <h3>4. Install Dependencies</h3>
        <pre>
pip install -r requirements.txt
        </pre>
        <h3>5. Run the Application</h3>
        <pre>
python app.py
        </pre>
        <p>
            After running the application, open your browser and navigate to 
            <a href="http://127.0.0.1:5000" target="_blank">http://127.0.0.1:5000</a>.
        </p>
        <h2>Technologies Used</h2>
        <ul>
            <li>Flask - Backend framework for web application</li>
            <li>Google Generative AI - AI models for generating responses</li>
            <li>Python-dotenv - For managing environment variables</li>
            <li>Gemini Models - For image analysis</li>
        </ul>
